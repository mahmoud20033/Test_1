‚öôÔ∏è ENVIRONMENT VARIABLES EXPLAINED
================================================================================

REQUIRED:
MONGODB_URI     ‚Üí mongodb+srv://user:pass@cluster.mongodb.net/db
NODE_ENV        ‚Üí "production"
PORT            ‚Üí 8080 (or cloud provider default)

RECOMMENDED:
ALLOWED_ORIGINS ‚Üí https://yourdomain.com
JWT_SECRET      ‚Üí Your random secret key

HOW TO GET MONGODB_URI:
1. Go to mongodb.com/cloud/atlas
2. Create cluster (M0 free)
3. Cluster ‚Üí Connect ‚Üí Connect your application
4. Copy connection string
5. Replace <password> with your user password
6. Result: mongodb+srv://admin:pass123@cluster0.mongodb.net/mydb

  $ git add .
  $ git commit -m "Production ready"
